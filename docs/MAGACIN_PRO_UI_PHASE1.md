# MAGACIN PRO UI PHASE 1

Urađeno u ovoj verziji:

- Novi ERP/WMS layout za magacin rolni
- Sređen spacing i grid
- Modernizovana tabela rolni
- Status badge boje: Na stanju, Rezervisano, Delimično, Iskorišćeno
- Hover efekti na karticama i tabeli
- Novi dashboard cards za broj rolni, metre, kg, rezervacije, tipove materijala i lokacije
- Bolji pregled po materijalu i širini
- Profesionalniji panel za unos rolni / prijem / packing list
- Novi desni detalj panel rolne
- Priprema za QR workflow, parent/child role, skidanje metraže, rezervacije i istoriju
- Build proveren: npm run build prolazi

Napomena: poslovna logika nije menjana. Ovo je UI/WMS faza sređivanja.
