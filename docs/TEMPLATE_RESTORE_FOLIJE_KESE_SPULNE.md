# TEMPLATE RESTORE — Folije / Kese / Špulne

Vraćen je ProductTemplateEngineV20 sa dobrim template ekranima za:
- Folije
- Kese
- Špulne

Zadržano:
- Material PRO tabela
- bez checkbox-a Žuta
- samo Š i L
- postojeća struktura sekcija za folije, kese i špulne

Napomena: template strukturu više ne brisati; ubuduće samo dorađivati UI i logiku unutar postojećih sekcija.
