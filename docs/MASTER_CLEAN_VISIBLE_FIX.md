# MASTER CLEAN — VISIBLE TEMPLATE FIX

Ova verzija uvodi vidljivu izmenu u Baza proizvoda / Template kartici:

- template nema cenu materijala
- materijali su u jednoj liniji: Vrsta / Oznaka / Debljina / Širina / Koeficijent / Napomena
- proizvod se čuva kao kartica u Bazi proizvoda
- unutar proizvoda postoje tabovi:
  - Osnovni podaci
  - Materijali
  - KPDF / dokumenti
  - Potreba materijala
  - Istorija / nalozi
- jasno je prikazano da se konkretna rola bira tek u fazi Potreba materijala
- u Potrebi materijala su pripremljena polja za QR, lokaciju, šaržu, LOT i rezervaciju

Važno: kalkulacije i cene nisu dirane.
