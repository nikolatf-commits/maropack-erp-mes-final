# Material PRO tabela — unified kalkulacije

Urađeno:
- Folija je uzeta kao referentni prikaz materijala.
- Kalkulacija kese koristi istu Material PRO tabelu kao folija.
- Kalkulacija špulne koristi istu Material PRO tabelu kao folija.
- Izbačena je kolona/checkbox Žuta.
- Ostaju samo Š = Štampa se i L = Lakira se.
- Build je testiran: npm run build prolazi.
