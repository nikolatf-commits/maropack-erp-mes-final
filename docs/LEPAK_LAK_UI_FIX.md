# LEPAK I LAK UI FIX

- Uklonjeni veliki plavi/zeleni ovalni okviri.
- Lepak i lak su sada prikazani kao profesionalna tabela.
- Kolone: Tip, Potrošnja kg/m², Prolazi, Cena €/kg, Ukupno kg / 1000m.
- Material PRO tabela ostaje jedinstvena za folije/kese/špulne.
- Build je prethodno proveren i prolazi.

Napomena za poređenje sa Excelom:
Za 100% identičan rezultat potrebno je da u aplikaciji i Excelu budu isti ulazni podaci i iste formule. Na screenshotu je app prikazivala 83.90 €, a Excel 83.945 €, razlika je u decimalama/ulaznom parametru lepka ili zaokruživanju.
